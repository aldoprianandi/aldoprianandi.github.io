About

I turn repeated work into tools.

My strongest work is not code for code's sake. It is noticing a manual process, making it easier to run, and packaging the result so it can survive another day.

01.

What I do now.

I build AI-assisted internal utilities, browser automations, content workflows, and small dashboards. The common thread is simple: repeated task, clearer system.

I use AI as leverage, then I verify behavior through real runs, screenshots, command output, and documentation. Fast only matters when the result is still usable.

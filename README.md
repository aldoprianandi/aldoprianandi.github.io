# Aldo Prianandi

Static personal website for [aldoprianandi.github.io](https://aldoprianandi.github.io).

The site is intentionally small: plain HTML, one CSS file, and a single logo asset. It presents Aldo's current public positioning around internal tools, browser automation, content operations, and AI-assisted workflow building.

## Local Preview

```bash
npx serve .
```

There is no build step. GitHub Pages serves the files directly from the repository.

## Structure

- `index.html`: home and current positioning
- `about.html`: capabilities, tools, and background
- `projects.html`: selected work, Cinethetic, Bolakokgitu, and operational case studies
- `css/workbench-layout-20260507.css`: visual system and responsive layout
- `assets/logo.png`: site mark
- `assets/cinethetic/`: compressed public demo previews from Cinethetic

## Repository Notes

This repo is the public website. It keeps the implementation simple and points
to selected work instead of mirroring every experiment.
